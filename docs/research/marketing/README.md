# docs/research/marketing/ — positioning, promotion, pricing

Marketing and go-to-market research for the SaaS, kept to the same standard as build research:
live sources, weighted options, a recorded decision. Driven by the `saas-research` skill.

## What goes here

| Topic | Example file |
|---|---|
| Positioning & messaging | `positioning.md` |
| Competitor teardown | `competitors-<name>.md` |
| SEO / keyword angles | `seo-keywords.md` |
| Launch channels (Product Hunt, HN, Reddit, dev.to, X/LinkedIn) | `launch-plan.md` |
| Pricing & packaging | `pricing.md` |
| Landing-page conversion patterns | `conversion.md` |

Each doc ends in a recommendation a human can act on. For deeper multi-source reports, the host
`deep-research` skill and the `blog` skill suite are available — see `saas-research`.
